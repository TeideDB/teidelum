Web REPL

Browser-based SQL console for interactive data exploration.

Overview

  • Browser-based SQL console served locally
  • CodeMirror 6 editor with SQL syntax highlighting and autocomplete
  • Virtual-scrolling result tables for large datasets
  • Built-in JavaScript console
  • Dark theme matching the Teide color palette

Prerequisites

  • Node.js >= 18
  • CMake >= 3.15, C17/C++17 compiler
  • git

Installation

git clone https://github.com/TeideDB/teide-js.git
cd teide-js
npm install
npm run build

Running

npm run repl               # Start server (manual browser open)
node bin/teide.js          # Start server + auto-open browser

Opens at http://127.0.0.1:3141 (auto-increments if port busy).

Interface Guide

Editor

  • SQL editor with syntax highlighting (one-dark theme)
  • Autocomplete triggers as you type (2+ characters), Tab to accept
  • Ctrl+Enter (Cmd+Enter on Mac) to execute
  • Green "Run query" button in bottom-right corner
  • SQL tabs — double-click to rename

Sidebar

  • Tables panel: expandable tree showing table names, row counts, and column types
  • Filter input to search tables
  • History panel: clickable list of past queries
  • Toggle between panels via icon rail

Bottom Pane

Three tabs below the results:

  • Output — Query result table with virtual scrolling, column headers with type badges, numbers right-aligned, Download CSV/JSON/Copy
  • Log — Scrollable history of all queries with timestamps and timing
  • Console — JavaScript REPL (eval any JS, arrow keys for history)

Loading Data

.load /path/to/data.csv
.load /path/to/data.csv as myTable

Or via SQL:

CREATE TABLE t AS SELECT * FROM read_csv('/path/to/data.csv');

Querying

SELECT * FROM sales;
SELECT category, SUM(price) FROM sales GROUP BY category;
SELECT * FROM sales WHERE price > 100 ORDER BY price DESC;

Dot-Commands

CommandDescription
.helpShow available commands
.tablesList registered tables
.schema <table>Show column names and types
.load <file> [as <name>]Load CSV file
.save <table> <file>Export table to CSV
.timer on|offToggle query timing
.clearClear results

Architecture

A Node.js HTTP server serves a single HTML page. WebSocket handles bidirectional communication for queries, autocomplete, and metadata. The native Teide engine executes queries via C++ NAPI bindings.
